# Termux-Packet-Sniffer
Terminal based network packet sniffer for Termux and Linux. Capture, parse, and analyze real-time IP, TCP, UDP, ICMP, DNS, and HTTP traffic via an interactive Curses TUI. Features built-in anomaly detection, traffic filtering, and session exports to PCAP, JSON, and CSV. Lightweight, zero-bloat, and optimized for mobile terminal environments.
